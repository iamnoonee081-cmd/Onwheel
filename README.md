# SproutsOnWheel Landing Page

A modern, responsive landing page for SproutsOnWheel - a nutrition delivery service for gym enthusiasts.

## Project Structure

```
sproutsLandingPage/
├── index.html              # Main HTML file
├── css/
│   └── styles.css          # All CSS styles
├── js/
│   ├── animations.js       # GSAP animations
│   └── three-background.js # Three.js background effects
├── assets/
│   └── images/            # Image assets (to be added)
└── README.md              # Project documentation
```

## Features

- **Responsive Design**: Mobile-first approach with CSS Grid and Flexbox
- **Smooth Animations**: GSAP-powered scroll animations and interactions
- **3D Background**: Three.js particle system for visual appeal
- **Modern CSS**: CSS custom properties, modern selectors, and animations
- **Performance Optimized**: Preconnected fonts, optimized animations

## Technologies Used

- HTML5 semantic markup
- CSS3 with custom properties and modern features
- JavaScript ES6+
- GSAP (GreenSock Animation Platform)
- Three.js for 3D graphics
- Google Fonts (Inter)

## Getting Started

1. Open `index.html` in a web browser
2. For development, use a local server (e.g., Live Server extension in VS Code)

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)
